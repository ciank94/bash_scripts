This repository contains bash aliases and scripts mainly used to aid in navigation of sinmod.
-gen_bash: Creates empty bash scripts with bash compiler in preamble, creates file in bash_scripts folder and turns on permission for all users.
-girem: Adds remote directory and local directory name with prompts from user.
-insi: Creates an init file with settings to manipulate the trackfile and simSettings for easier simulation initializations.
-sfsin: search for files with text pattern in libf90 and gin
-stsin: search for text within sinmod files in libf90 and gin
-bash_aliases.txt: A record of the bash aliases created
-git_log.txt: A record of file edits
-emac_aliases.txt: A record of emac aliases created
